A) Installation - top

1. Upload zipped file Training.zip to your server directory and unzip there.
2. Create a database.
3. Go to the Database file application/config/database.php and fill up with your database details.

$db['default']['hostname'] = 'Host Name';
$db['default']['username'] = 'Database Username';
$db['default']['password'] = 'Password';
$db['default']['database'] = 'Database Name';
4. Import the sql file into your database from the Database folder.
5. Upload the .htaccess file from htaccess folder into the server directory.

Its Done !!!!!

B)Modules - top

Through this Training Center Management system User can manage all kind of management and financial activities of a Training Center. In this system there are modules like Course, Batch, Students, Attendence, Routine, Payment, Staff Attendence Tracking, Task Management, Events etc. In a summary you need to use this software through this flow create course -> create batch ->create students ->all other activities

C) Course - top

Go to Add Course from Course module in sidebar.
Fill the form with necessary info. Submit.

Courses will be enlisted here one by one.

New curse also can be created by clicking on 'add curse' button in All Courses page.

D)Batch - top

Go to Batch-> Add Batch in the side menubar
Fill The form
Then Submit.
Batch will be created.

Through this module user can manage all the batches and related issues.

E) People - top

Students, Instructor and Employees can be created through this module.
In Student section you can find the details button. After clicking that you will find the every detail of a student. Students course, batch and payment information are there.

F) Routine - top

After Creating the courses and batches you can create routine for batches.

Steps:
1. Go to Routine --> Add Routine.
2. Select Course and Batch.
3. Select Weekday by checkbox and then input the time.
4. Submit.

G)Students Attendence - top

Go To Attendence -> Add Attendence Select Course and Batch.
Click Next Button.
After that you will find all the student of that batch and select option for there attendence. Just select thier attendence status and submit.

H)Staff Attendence - top

When Staffs (Instructors, Employees) login to profile they will find a button for sign in.
Server will automatically detect the time and Ip Address. And a sign out button will appear
After work they will just sign out the system
Admin can see the attendence report from Attendence -> Staff Attendence option in menubar.

I)Finance - top

Payment Can be Done from Finance -> Add Payment Expense Can be Done from Finance -> Add Expense From Reporting Module You can check Full, Course-Wise and Batch-wise Financial Reports.

J)Tasks, Events & Notice - top

You can manage Tasks and Task Reporting from task Module. Event module manage events with a functional calendar. There is a notice board in the Dashboard. You can manage those notices from notice module

K) Settings - top

Site Configuration Can be changed through this module.

Different types of settings can be managed.
1. Organization Name.
2. Address
3. Phone Number
4. Email Address.
5. Currency.
6. Date Format.
7. Login Title.
8. Login Logo.
